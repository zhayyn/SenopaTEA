# Skill Exec: Generate Code

## Objective
Act as `@engineer` to write physical code strictly based on the approved specification.

## Execution Steps
1. **Read_Contract**: Open and parse `production_artifacts/Technical_Specification.md`.
2. **Scaffold_Structure**: Build the required folder structure and core application files (Frontend & Backend).
3. **Execute_Code**: Write the actual logic in the specified languages/frameworks.
4. **Commit_To_Build**: Save all raw code, ensuring `package.json` or `requirements.txt` are included, directly inside `app_build/`.
