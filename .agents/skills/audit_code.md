# Skill Exec: Audit Code

## Objective
Act as `@qa` to ensure the generated codebase is perfectly functional and secure.

## Execution Steps
1. **Assess_Alignment**: Cross-reference the code in `app_build/` against `Technical_Specification.md`.
2. **Hunt_Vulnerabilities**: Scan for dependency mismatches, syntax errors, and unhandled promises.
3. **Patch_And_Overwrite**: Fix the identified issues and overwrite the flawed files in `app_build/` with the polished revisions.
